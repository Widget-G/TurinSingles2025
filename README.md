# TurinSingles2025

## Instrukcja subskrypcji w Apple Calendar

1. Wgraj repozytorium na GitHub: `https://github.com/Widget/TurinSingles2025`
2. Włącz GitHub Pages (Settings → Pages → Branch: main, folder: / (root))
3. Twój plik będzie dostępny pod linkiem:
   `https://widget.github.io/TurinSingles2025/ATP_Finals_2025_Singles_Turin.ics`
4. W Apple Calendar (macOS/iOS):
   - Plik → Nowa subskrypcja kalendarza
   - Wklej link zamieniając https:// na webcal://
     `webcal://widget.github.io/TurinSingles2025/ATP_Finals_2025_Singles_Turin.ics`
   - Ustaw automatyczne odświeżanie (np. co dzień)
5. Po ogłoszeniu meczów ATP, zaktualizuj plik `.ics` w repozytorium, a Apple Calendar pobierze zmiany automatycznie.
